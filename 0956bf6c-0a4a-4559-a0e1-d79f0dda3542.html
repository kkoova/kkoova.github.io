Уровень 10: Работа с сетью

В этом задании ты познакомишься с базовыми командами работы с сетью в Linux: ping, netstat и ifconfig.

  • ping 8.8.8.8 — проверка доступности внешнего сервера
  • netstat -tuln — показать все открытые порты и слушающие сервисы
  • ifconfig — посмотреть IP-адреса интерфейсов

Шаги задания:

  1. Перейди в папку уровня 10:
    2. cd ~/kali_quest/level10
  2. Попробуй проверить сеть с помощью ping:
    3. ping -c 4 8.8.8.8
  3. Посмотри открытые порты:
    4. netstat -tuln
  4. Посмотри свой IP-адрес:
    5. ifconfig
  5. Открой файл с флагом:
