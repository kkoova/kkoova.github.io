Уровень 2: Работа с файлами

В этом задании ты освоишь базовые команды работы с файлами в Linux:

  • touch — создать новый файл
  • rm — удалить файл
  • ls — посмотреть содержимое папки
  • cat — открыть файл и прочитать его содержимое

Шаги задания:

  1. В домашней папке квеста открой каталог для уровня 2: 
    cd ~/kali_quest/level2
  2. Создай новый файл (например, myfile.txt): 
    touch myfile.txt
    Проверьте в проводнике
  3. Удалите временный файл: 
    rm temp_file.txt
    Проверьте в проводнике
  4. Найди файл с флагом и прочитай его: 
    cat flag.txt
  5. Напиши текст флага и введи его в форму ниже.
