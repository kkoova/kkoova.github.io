Уровень 9: Работа с пакетами

В этом задании ты познакомишься с базовыми командами управления пакетами в Linux: apt и dpkg.

  • sudo apt update — обновление списка пакетов
  • sudo apt install package_name — установка пакета
  • dpkg -l — список установленных пакетов
  • dpkg -s package_name — информация о пакете
