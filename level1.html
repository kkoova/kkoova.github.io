Уровень 1: Навигация в Linux

В этом задании ты познакомишься с базовыми командами Linux, которые помогают ориентироваться в файловой системе.

  • ls — показать список файлов и папок в текущей директории
  • ls -a — показать все файлы, включая скрытые
  • cd — сменить каталог
  • pwd — показать текущий путь
  • cat — открыть файл и прочитать его содержимое

Шаги задания:

  1. Скачай и запусти скрипт для уровня 1 (если ещё не сделал):
    2. bash <(curl -s https://kkoova.github.io/setup.sh)
  2. Перейди в папку уровня 1:
    3. cd ~/kali_quest/level1
  3. Посмотри, что находится в папке, включая скрытые файлы:
    4. ls -a
  4. Перейди в скрытую папку:
    5. cd .hidden_folder
  5. Открой файл с флагом:
    6. cat flag.txt
  6. Скопируй текст флага и введи его в форму ниже.
