Тестирование веб-приложения на проникновение методом полного перебора - Тренировка

1. Цель работы

Закрепить полученые в 6 лаб навыки тестирования веб-приложения на устойчивость к атакам методом полного перебора (brute-force).

Содержание отчета

  • Титульный лист
  • Задание
  • Тренировка
Внимание! За несвоевременное представление работы на проверку оценка снижается на 1 балл!

2. Начало работы

! ВНИМАНИЕ: DVWA не эмулирует уязвимости веб-приложений, уязвимости внутри DVWA являются настоящими, поэтому работайте с ним только на виртуальной машине. Не загружайте его на любой сервер, имеющий выход в Интернет.

Шаг 1. Запустите приложение DVWA

sudo docker run docker.io/vulnerables/web-dvwa:latest

Отчет офорляйте по типу:
1. Команда для нахождения пароля
2.Скриншот правильного выполнения на данной странице
